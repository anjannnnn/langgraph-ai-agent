\# LangGraph AI Agent



\## Project Overview

This project is an AI-powered agent built using LangGraph that analyzes user input and generates intelligent responses using LLM workflows.



\## Features

\- Emotion analysis from text

\- AI workflow using LangGraph

\- Modular node-based architecture

\- Fast inference using Groq API



\## Technologies Used

\- Python

\- LangGraph

\- LLM

\- Groq API

\- dotenv



\## Installation



1\. Clone the repository

git clone https://github.com/yourusername/langgraph-ai-agent.git



2\. Install dependencies

pip install -r requirements.txt



3\. Create a .env file

Add your API key:



GROQ\_API\_KEY=your\_api\_key\_here



4\. Run the project

python main.py



\## Example Output



Input:

"I feel stressed today"



Output:

Detected Emotion: Stress

AI Response: Try taking a short break and relaxing your mind.
